## About the App

This project is an authentication application built with Express.js, featuring user login and sign-up functionalities, deployed on a DigitalOcean server.

---

## Features

- User registration (sign-up)
- User login with session or token-based authentication
- Basic validation and error handling

---

## Usage

Send HTTP requests to the server endpoints to register new users and log in existing users.
